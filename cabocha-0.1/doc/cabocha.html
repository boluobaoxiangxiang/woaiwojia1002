NAME

cabocha - manual page for cabocha of 0.1  

SYNOPSIS

cabocha [optsions] <files>...  

DESCRIPTION

CaboCha (Yet Another Japanese Dependency Structure Analyzer)

-t, --action-mode=TYPE
set action mode 0 - parser mode (default) 1 - training mode
-f, --output-format=TYPE
set output format type 0 - tree (default) 1 - lattice 2 - tree+lattice
-I, --input-layer=LAYER
set input layer 0 - raw sentence layer (tagged with ChaSen, default) 1 - POS tagged layer 2 - POS tagged and Chunked layer 3 - POS tagged, Chunked and Feature selected layer
-O, --output-layer=LAYER
set ouput layer 1 - POS tagged layer (tagged with ChaSen) 2 - Chunked layer 3 - Chunked and Feature selected layer 4 - Parsed layer (default)
-m, --parser-model=FILE
use FILE as parser model file
-M, --chunker-model=FILE
use FILE as chunker model file
-C, --candidate
partial parsing mode
-r, --rcfile=FILE
use FILE as rc file (default: /usr/local/etc/cabocharc)
-c, --chasenrc=FILE
use FILE as chasenrc
-o, --output=FILE
set the output file name
-v, --version
show the version of CaboCha and exit.
-h, --help
show this help and exit.
